# OpenClaw 十大宝典

AI助手生活操作系统十大核心技能，覆盖日常生活全场景。

## 宝典列表

| # | 名称 | 技能目录 | 功能 |
|---|------|---------|------|
| 1 | 小红书自动发布官 | `xiaohongshu-publisher` | 拍照+一句话→自动出标题文案→自动发布小红书 |
| 2 | 每日新闻汇编官 | `news-digest-operator` | 每天定时推送10条干货早报 |
| 3 | 生命体征监测 | `alive-risk-monitor` | 独居24h安全守护，异常自动通知紧急联系人 |
| 4 | 账单财务军师 | `bill-financial-advisor` | 截图识别账单，自动分析消费+规划预算 |
| 5 | 论文格式整理师 | `financial-experiment-writer` | 自动排版+参考文献格式+标准Word输出 |
| 6 | 下载文件夹净身大师 | `downloads-folder-organizer` | 一键扫描归类，自动整理成分类文件夹 |
| 7 | 冰箱清道夫 | `fridge-inventory-manager` | 拍照录入食材→临期提醒→智能菜谱推荐 |
| 8 | 面试模拟战 | `product-manager-interviewer` | 简历诊断+压力面试+评分复盘 |
| 9 | 旅行行李官 | `travel-packing-officer` | 输入目的地→自动生成打包清单 |
| 10 | 影剧看门人 | `movie-finder` | 搜片+多源评分+平台查询+私人观影榜单 |

## 运行环境

- [OpenClaw](https://github.com/openclaw/openclaw) — AI助手框架
- 每个技能目录包含独立的 `SKILL.md` 定义文件和可执行脚本

## 使用方式

将对应技能目录放入 OpenClaw 的 `skills/` 目录下即可激活。
