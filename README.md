## 🎯 这是什么？

把 AI 助手从"会聊天的机器人"升级为"能干活的生活管家"。

十大宝典覆盖了日常生活中最常见的十个场景 —— 从早上看新闻、做饭管冰箱，到出门旅行、找工作面试，再到发小红书赚钱 —— 全部自动化。

> 💡 **核心理念**：你只需要一句话，剩下的交给 AI。

---

## 📖 十大宝典

### 1️⃣ 小红书自动发布官 `xiaohongshu-publisher`

> 📸 拍照 + 一句话 → 自动生成标题文案 → 一键发布

你随手拍张照片，说一句"发个种草笔记"，AI 就会：
- 自动识别图片内容
- 生成小红书风格标题和正文
- 配好标签
- 直接发布到你的账号

**场景**：吃饭拍个照 → "帮我发小红书" → 3秒完成

---

### 2️⃣ 每日新闻汇编官 `news-digest-operator`

> 📰 每天早上 7:00，10条最有价值的新闻准时推送

不用自己刷热搜了。AI 会自动搜索、去重、总结，每天准时把精华推送到你手机上。

**场景**：早起洗漱 → 看手机 → 今日要闻已备好

---

### 3️⃣ 生命体征监测 `alive-risk-monitor`

> ❤️ 独居 24h 安全守护，异常自动通知紧急联系人

每天检测一次"我还活着"信号，超过 24 小时没响应就自动发邮件给紧急联系人。

**场景**：一个人住 → 设置心跳 → 家人安心

---

### 4️⃣ 账单财务军师 `bill-financial-advisor`

> 💰 截图识别账单，自动分析消费 + 规划预算

导出你的消费记录（CSV/截图），AI 秒级完成：
- 消费分类统计
- 异常消费识别
- 下月预算建议

**场景**：月底发愁钱花哪了 → 发个账单截图 → 完整分析报告

---

### 5️⃣ 论文格式整理师 `financial-experiment-writer`

> 📝 自动排版 + 参考文献格式 + 标准 Word 输出

告诉 AI 论文题目，直接生成格式规范的 Word 文件。模板、字号、行距、参考文献全部搞定。

**场景**：DDL 前一天 → "帮我写一篇XX论文" → Word 文件到手

---

### 6️⃣ 下载文件夹净身大师 `downloads-folder-organizer`

> 📁 一键扫描归类，乱七八糟 → 整整齐齐

自动扫描你的下载文件夹，按文件类型（文档/图片/视频/安装包...）+ 年/月重新归类整理。

**场景**：下载文件夹800个文件 → 一键整理 → 清爽分类

---

### 7️⃣ 冰箱清道夫 `fridge-inventory-manager`

> 🥦 拍照录入食材 → 临期提醒 → 库存管理

对着冰箱拍张照，AI 自动识别食材、记录库存、推算保质期。快过期的会提前提醒你。

**场景**：逛完超市拍照 → 自动建库存表 → 不再浪费食物

---

### 8️⃣ 面试模拟战 `product-manager-interviewer`

> 🎤 简历诊断 + 压力面试 + 评分复盘

上传你的简历，AI 会：
- 诊断简历的优缺点
- 模拟真实面试官提问
- 逐步追问（压力面试风格）
- 最后给出改进建议

**场景**：明天面试 → 上传简历 → 今晚模拟三轮

---

### 9️⃣ 旅行行李官 `travel-packing-officer`

> ✈️ 输入目的地 + 日期 → 完整打包清单 + 天气预警

告诉 AI 你要去哪、什么时候去、怎么去，它会查天气、分析穿搭需求，生成详细清单。

**场景**："后天飞哈尔滨" → 厚羽绒服+暖宝宝全提醒到了

---

### 🔟 影剧看门人 `movie-finder`

> 🎬 搜片 + 多源评分 + 平台查询 + 私人观影榜单

想看什么直接问，AI 会同时查豆瓣和 OMDb 的评分，告诉你哪个平台能看，还可以维护你的私人观影记录。

**场景**："最近有什么好电影" → 推荐列表 + 评分 + 观看平台

## 🏗️ 技术架构

```
openclaw-ten-arts/
├── xiaohongshu-publisher/    # 宝典一
├── news-digest-operator/     # 宝典二
├── alive-risk-monitor/       # 宝典三
├── bill-financial-advisor/   # 宝典四
├── financial-experiment-writer/  # 宝典五
├── downloads-folder-organizer/   # 宝典六
├── fridge-inventory-manager/     # 宝典七
├── product-manager-interviewer/  # 宝典八
├── travel-packing-officer/       # 宝典九
├── movie-finder/                 # 宝典十
│   ├── SKILL.md              # 技能定义文件
│   ├── scripts/              # 可执行脚本
│   └── references/           # 参考文档
└── README.md
```

每个技能都是一个独立的模块，包含：
- `SKILL.md` — 技能定义和行为规则
- `scripts/` — 自动化脚本（Python）
- `references/` — 参考文档和配置模板

## 🚀 快速开始

### 1. 安装 OpenClaw

```bash
npm install -g openclaw
openclaw init
```

### 2. 添加宝典技能

```bash
# 克隆本仓库
git clone https://github.com/PBBEH/openclaw-ten-arts.git

# 将技能目录复制到 OpenClaw skills 目录
cp -r openclaw-ten-arts/*/ ~/.openclaw/workspace/skills/
```

### 3. 开始使用

```
你：帮我发个小红书笔记
AI：好的，请拍照或告诉我内容...

你：整理一下我的下载文件夹
AI：正在扫描 ~/Downloads ...

你：分析一下这个月的账单
AI：请发送你的账单文件...
```
