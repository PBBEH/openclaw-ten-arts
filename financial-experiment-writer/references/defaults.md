# 默认模板与生成约定

## 默认模板
- 绑定模板：`assets/default-template.docx`
- 来源：`/Users/yaoyi/Desktop/实验论文模板.docx`
- 当前默认用途：金融实验报告

## 默认写法
- 直接按学校实验报告格式输出
- 默认生成四部分：实验题目、实验目的和要求、实验内容、实验总结
- 默认口吻：本科生实验报告，可直接提交，再按需要微调

## 触发方式
- 用户提到“宝典五”
- 用户给出 `.docx` 模板并要求按模板写实验/课程/论文报告
- 用户要求“用这个模板生成一份报告并发给我”

## 产出要求
- 优先生成 `.docx`
- 默认保留模板中的页眉页脚、封面与版式
- 不额外发明不存在的实验数据；若用户没给数据，可先写通用版实验报告
